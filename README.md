# 建築施工メディエーター（仮）サイトモック

建築施工メディエーター（仮）の構想を、実際の公開サイトを想定した文言と多ページ構成で整理した静的モックです。
GitHub Pages で公開しやすいよう、依存なしの HTML / CSS のみで構成しています。

## ページ構成

- `index.html`: トップページ
- `about.html`: 資格について
- `program.html`: 資格取得の流れ
- `profile.html`: 代表メッセージ
- `faq.html`: FAQ
- `contact.html`: お問い合わせ

## ローカル確認

1. `python3 -m http.server 8000`
2. ブラウザで `http://localhost:8000` を開く

## GitHub Pages

`main` ブランチの `/(root)` を公開対象に設定すると、そのまま表示できます。

## 差し替えが必要な項目

- 問い合わせ先メールアドレス
- 代表者プロフィールの固有情報
- 正式な講習開始時期や認定方法
- 正式名称と英語表記
