# 建築施工メディエーター（仮）共有モック

新資格「建築施工メディエーター（仮）」の構想を、クライアントに共有するための静的サイトです。
GitHub Pages にそのまま載せられるよう、依存なしの `index.html` と `styles.css` のみで構成しています。

## 使い方

1. ローカル確認: `python3 -m http.server 8000`
2. ブラウザで `http://localhost:8000` を開く
3. GitHub Pages 利用時はリポジトリのルートを公開対象に設定

- 想定活用シーン
